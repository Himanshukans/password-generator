# Neon PassGen ⚡

A high-performance, secure password generation tool built with **React** and **Tailwind CSS**. This project features a premium dark-themed interface with vibrant neon accents.

## ✨ Features
- **Dynamic Length Control:** Adjust password length from 6 to 50 characters using a slider or precision Plus/Minus buttons.
- **Custom Security:** Toggle between including Numbers and Special Characters.
- **Real-time Strength Indicator:** Visual feedback on password security levels (Weak to Very-Hard).
- **One-Click Copy:** Instant clipboard integration with a "Copied" status animation.
- **Modern UI:** Built with a "premium-dark" aesthetic, featuring CSS glows and gradients.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Hooks used:** `useState`, `useEffect`, `useCallback`, `useRef`
- **Build Tool:** Vite

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
